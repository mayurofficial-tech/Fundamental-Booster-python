# 🧾 Interactive Personal Data Collector

## 📌 Description
This is a beginner Python project that collects user input such as name, age, height, and favorite number, and displays the information along with data types and memory addresses.

It also calculates the approximate birth year of the user.

---

## 🚀 Features
- Interactive user input
- Displays data type using `type()`
- Shows memory address using `id()`
- Calculates approximate birth year
- Simple and clean output

## 🛠️ Tech Used
- Python (Basics)
